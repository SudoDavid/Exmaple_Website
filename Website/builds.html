Robotic Arm Control iOS Mobile Application

This mobile application featurtes wireless network connectivity via ios mobile device, amazon web services, and Arduino Yun Microcontroller.

The newest implementation of the robotic arm project was done as a Senior Undergraduate maintence project. Further this project now provides a front-end interface via iOS mobile device to meet modications.

Adaptive maintence involved upgrading from an Arduino Mega Microcontroller to the Arduino Yun Microcontroller. Thus providing additional ports, memory storage, and processing power. Corrective maintence including Amazon Networking and platform services. These services include; SQS, S3, Lamba, API Gateway, and IAM.
